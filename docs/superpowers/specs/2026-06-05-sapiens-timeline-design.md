# Sapiens Timeline — 设计规格书

**日期**: 2026-06-05  
**主题**: 《人类简史》交互式时间线网站

## 概述

单页交互式时间线网站，以"暗色星图"视觉风格呈现《人类简史》全书时间线。用户可缩放、平移星空画布，点击节点查看中英双语事件卡片。

## 架构

单文件 `index.html`，内嵌 CSS + JS。Canvas 2D 绘制星空与时间线，HTML overlay 显示 UI 控件和详情卡片。

### 层次结构

1. **Canvas 全屏层** — 三层星空粒子(远/中/近视差) + 时间线节点 + 星云区域 + 星座连线
2. **HTML 叠加层** — 时代标签栏、详情卡片、迷你地图、底部搓条、缩放控件

### 渲染层

- **远层星场**: 静态微星点，约 400 颗
- **中层星场**: 缓闪星座群，约 200 颗，周期性明暗
- **近层星场**: 漂移大光点，约 50 颗，极慢移动
- **节点层**: 星光节点 (significance 决定大小和动画)
- **星云层**: 四大时代区域背景色晕 (径向渐变)
- **连线层**: 节点间隐约细线，仅在近景显示

## 交互

| 操作 | 行为 |
|------|------|
| 滚轮 / 双指捏合 | 缩放 (0.3x ~ 4.0x) |
| 拖拽 / 单指滑动 | 平移画布 |
| 点击节点 | 弹出详情卡片，画布自动居中 |
| 卡片外 / ESC | 关闭卡片 |
| 卡片内 ← → | 上/下一个节点 |
| 底部搓条 | 快速跳转时代 |
| 迷你地图 | 全局缩略 + 视口指示 |
| 键盘 1-4 | 跳转四大时代 |

### 缩放层级

| 层级 | 范围 | 显示内容 |
|------|------|----------|
| 远景 | < 0.5x | 四大星云区 + 时代标签 |
| 中景 | 0.5x ~ 1.5x | 关键节点 (significance=key) |
| 近景 | 1.5x ~ 3.0x | 所有节点 + 连线 |
| 最细 | > 3.0x | 节点标题文字、年份标签 |

默认起点：中景，居中认知革命区域。
