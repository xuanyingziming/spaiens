# spaiens
An interactive digital corridor that brings millions of years of human evolution to life on your screen.

An interactive data visualization website designed to take users on a visual journey through the grand narrative of human history—from the Big Bang, the origins of life, and human evolution, to the rise of modern civilization.

✨ Features
Immersive Timeline: Breaks away from tedious text with smooth interactions and visual effects to present the depth of history.

Multi-dimensional Perspectives: Covers pivotal historical epochs including the Cognitive Revolution, the Agricultural Revolution, the Unification of Humankind, and the Scientific Revolution.

Modern Tech Stack: Built on a high-performance frontend architecture to ensure fluid scrolling and responsiveness even with large volumes of historical data.

🛠️ Tech Stack
Frontend Framework: [e.g., React / Vue 3 / Next.js]

Visualization & Animation: [e.g., Tailwind CSS / GSAP / Three.js / Canvas]

Build Tools: [e.g., Vite / Webpack]
